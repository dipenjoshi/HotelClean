Frequently Asked Questions

What is this tool?

Housekeeping room list shares updates, room list between front office and housekeeping team. Makes tracking of each room status easier.

How to use?

1. Start by adding a Property ID using the + button. The property ID given can be shared with whole staff so they can access room list.

2. Add a housekeeper

3. Select date from the date picker. Default is today

4. Add rooms to the list

5. Update room statuses and notes as needed.

6. Pass on the property ID and website url to housekeeper.

7. Housekeeper will be able to see room list and update its status to clean. After selecting their name from employee dropdown.

Where is my data saved?

Your data is saved in cloud database. No personal information is stored.

How to export the data?

There is no way to export/import data. You can stop using it from tomorrow. There will be a sync option in timesheet tool later.
